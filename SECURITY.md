# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in darklens, please report it ronsibly.

**Do NOT open a public issue.**

Instead, email the maintainers directly or use GitHub's private vulnerability reporting.

We will rond within 48 hours and work on a fix.

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest | Yes |
| < latest | No |
